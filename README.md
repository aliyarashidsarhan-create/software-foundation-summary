This folder contains several files that summarize key software development concepts and Git operations:



Language comparisons (C#, Java, Python): Short summaries and comparison tables covering compiling behavior, speed, usage, advantages, and latest versions.



Tech stack overviews: Simple explanations of the MERN and .NET stacks, including frontend, backend, and database components.



Web development topics: Introductions to web development, databases, web applications, mobile applications, and desktop applications.



Agile \& SDLC: Clear explanations of Agile, Sprints, and the Software Development Life Cycle (SDLC).



Git commands \& workflows: Notes on common Git commands, including staging, committing, pushing, connecting remote repositories, resolving pull conflicts, and adding a README on GitHub.
